# 🐾 LazyCat

> Simple. Fast. Lazy.  
> Minimal CLI template generator written in C.

---

## 📦 About

LazyCat is a lightweight command-line tool that generates basic project templates for multiple programming languages.

No dependencies.  
No configuration.  
Just run and generate.
